Task 01: Superstore Sales Dashboard 📊
Project Overview
This project involved cleaning and analyzing a comprehensive Superstore Sales Dataset using Microsoft Excel. The goal was to transform raw sales data into an interactive dashboard that provides actionable insights into profitability and sales trends.

Key Objectives 🎯
Data Cleaning: Processed the dataset to ensure date consistency, currency formatting, and the removal of duplicates.

Pivot Table Analysis: Summarized Total Profit by Category and Sales Trends Over Time.

Interactivity: Integrated Slicers for Region and Segment to allow for dynamic data filtering.

Visual Storytelling: Developed a dashboard featuring Bar Charts for category comparison and Line Charts for temporal trends.

Key Insights 💡
Based on the analysis of the Superstore dataset:

Most Profitable Category: Technology consistently drives the highest profit margins compared to Furniture and Office Supplies.

Sales Seasonality: There is a significant upward trend in sales during the Q4 (Holiday Season), specifically in November and December.

Regional Performance: The West Region represents the highest sales volume across the dataset.

Performance Highlighting: Used Conditional Formatting to identify high and low-profit sub-categories, revealing that while Furniture has high sales, its profit margins are often the thinnest.
